# AI & Machine Learning Tasks 🤖

A collection of hands-on AI and Machine Learning projects built using Python and Scikit-learn. This repository demonstrates practical ML workflows, including data preprocessing, model training, hyperparameter tuning, and production-ready pipelines.

---

## Projects

| Project | Type | Models Used |
|---------|------|-------------|
| Churn Prediction Pipeline | Classification | Logistic Regression, Random Forest |
| Telco Churn ML Pipeline | Classification | Logistic Regression, Random Forest (Pipeline + GridSearchCV) |
| Multimodal Housing Price Prediction | Regression | Linear Regression, Random Forest |

---

## Churn Prediction Pipeline 📊

Predicts whether a customer is likely to churn based on customer information.

### Features
- Data preprocessing
- Missing value handling
- Feature encoding
- Model training and evaluation
- Classification metrics

---

## Telco Churn ML Pipeline 🔄

An end-to-end machine learning pipeline built using Scikit-learn's Pipeline API for customer churn prediction.

### Features
- Automated preprocessing using Pipeline
- Numeric scaling and categorical encoding
- Logistic Regression and Random Forest models
- Hyperparameter tuning using GridSearchCV
- Exported trained pipeline with Joblib
- Production-ready workflow

---

## Multimodal Housing Price Prediction 🏠

Predicts house prices using property-related features through regression models.

### Features
- Data preprocessing
- Feature engineering
- Regression model training
- Model performance evaluation
- Price prediction
